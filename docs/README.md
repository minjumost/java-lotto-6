    - [ ]  너무 상세하기 작성하지 않기(클래스와 함수 설계 등)

- [ ]  함수(메서드) 분리하기
    - [ ]  15라인을 넘어가지 않기

    - [ ]  indent depth가 3이 넘어가지 않도록 구현
- [ ]  테스트코드

- [ ]  하드코딩 하지 않기
- [ ]  Enum을 적용하기
